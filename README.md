# Salah Football Gala 2026

This is the official digital system for the Salah Football Gala 2026 tournament.

## Features

- Team Registration
- Player Registration with Photo
- Player ID Card with QR Code
- Paystack Payment Integration (MoMo Ghana)
- Admin Dashboard
- Season Control (2026)
- WhatsApp Notification
- Islamic + Modern Football Design
- Mobile Friendly
- Free Hosting Ready (Render)

## How to Use

1. Open the homepage: https://yourlink.onrender.com
2. Register Team
3. Register Player
4. Admin login: `admin` / `Salah2026`
5. Update Paystack secret key and WhatsApp number in app.py
